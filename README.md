# Algorithms Tasks 2026 - Birthday Cake Candles 

This repository contains the implementation and documentation for the "Birthday Cake Candles" algorithm project.

## Project Overview
The goal is to find the frequency of the tallest candle on a birthday cake given an array of candle heights. The project includes two algorithmic approaches:
1. **Iterative Approach (Non-Recursive):** $O(N)$ Time, $O(1)$ Space.
2. **Recursive Approach:** $O(N)$ Time, $O(N)$ Space.

## Files Included
* `BirthdayCakeCandlesIterative.java`: Contains the iterative logic.
* `BirthdayCakeCandlesRecursive.java`: Contains the recursive logic.
* `Documentation.pdf`: Comprehensive analysis, pseudo-code, and complexity comparisons.

## How to Run
Compile and run the Java files using any standard Java compiler or IDE. The programs feature an interactive console interface to input the number of candles and their heights.
